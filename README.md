# EQSpellParserToFile
Parse EQ Spells and save base and slots to gzipped text file\
\
To appropriately Parse, requires the following:\
data/dbstr_usL.txt - L Live, T Test, B Beta after i.e. dbstr_usL.txt for Live\
\
data/spells_usL.txt L Live, T Test, B Beta after i.e. spells_usL.txt for Live\
\
data/itemlist.txt - Items, you can get a public list from https://github.com/theriou/EQ-Item-List - this will be updated over time\
itemlist is in the ID^Name^StackSize format, if you wish to source elsewhere
